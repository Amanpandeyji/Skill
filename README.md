# SkillDev 

SkillDev is a simple and beginner-friendly educational website designed for college students to explore different technology stacks through curated learning resources and mind maps. The platform helps students learn efficiently using organized content and a clean user interface.

---

##  Features

- Curated learning resources
- Mind maps for better understanding
- Responsive user interface
- Smooth animations
- Beginner-friendly design
- Organized tech stack sections

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP Animation Library

##  How to Run the Project

### 1️ Clone the Repository

```bash
git clone https://github.com/Amanpandeyji/SkillDev.git
```

### 2️ Open Project Folder

```bash
cd SkillDev
```

### 3️ Run the Project

Simply open the `index.html` file in your browser.
